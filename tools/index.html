YouTube to MP3

Converted files will be stored in a newly created folder within the directory called "ConvertedSongs".


Download Requires dependencies Pytube and PySimpleGUI.

HTML Image Tag Generator

Selects a file and copies a generic HTML image tag to your clipboard including width, height, and an absolute path.


Download Requires dependencies PIL, Tkinter, and Pyperclip.
